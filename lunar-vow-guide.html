1. 角色定位

角色：德麗莎·阿波卡利斯 戰衣之一

屬性：異能（PSY） · 雷元素為主，附帶出血（Bleed）傷害

定位總結：可以當主 C，以雷傷 + 出血瘋狂鋸怪；亦可以當雷隊輔助，幫其他雷 C 開路、上易傷。

2. 操作核心：啃咬值 & 狂化

啃咬值（Biting Mark）

  • 普攻／部分技能會幫你累積啃咬值。
  • 啃咬值滿咗之後，使用武器技就可以進入 狂化狀態。

狂化狀態（Frenzy）

  • 攻擊變成電鋸多段 hit，極快疊出血和雷傷。
  • 期間有一定生存加成，方便你長時間站場。
  • 係呢個角色最爽、亦係最關鍵嘅輸出時間。

一句講晒：先累積啃咬 → 滿咗開武器技入狂化 → 狂化期間盡量打滿輸出，再用大招收尾。

3. 入門操作循環（大約手感）

  1. 用隊友先鋪好 buff（雷傷加成、易傷、聚怪之類）。
  2. 切入月下，可以用 QTE 入場或者直接切人。
  3. 用普攻／武器技疊啃咬值，見到差唔多滿就準備開狂化。
  4. 啃咬值滿 → 開武器技進入狂化，瘋狂按攻擊去鋸怪。
  5. 狂化尾段開大招，集中把傷害爆出嚟，之後再 decide 係咪切返隊友繼續輪轉。

4. 裝備方向（簡易版）

武器

  • 專武十字架：有就直接帶，為月下量身訂做，會加強雷傷、出血同整體輸出循環。
  • 過渡武器：揀高攻擊力、加 雷元素傷害 或 全傷 嘅十字架，永遠比亂用物理向好。

聖痕（Stigmata）

  • 專屬聖痕套：如果你係老婆廚，呢套基本上係目標，輸出同輔助能力都會被拉高。
  • 代用思路：優先考慮加雷傷、全傷、易傷、攻擊、爆擊／爆傷嘅套裝，盡量唔好擺防禦向聖痕喺主 C 位。

5. 配隊方向（入門思路）

標準雷隊

  • 主 C：月下誓約・予愛以心。
  • 雷隊輔助：提供雷傷 buff、易傷／減防、聚怪等效果嘅角色。
  • 第三位：可以係補 buff、補聚怪、或者提供額外輸出嘅角色。

簡單講：先用兩個隊友鋪好 buff → 再切月下開狂化爆發 → 收尾再切人。

6. 小貼士

  • 狂化要揀 timing：唔好一滿啃咬就即刻開，最好等 buff 齊、怪又聚好先一口氣爆發。
  • 大招當收尾技：狂化後期開大，可以令成個循環嘅輸出集中爆出嚟，畫面同體感都會好舒服。
  • 記得佢都係雷輔：有啲關可以先用月下開大幫雷隊上易傷，再切返另一隻雷主 C 收頭。
  • 資源分配：如果你係月下本命玩家，建議優先升武器 → 聖痕 → 再慢慢補其他角色。
